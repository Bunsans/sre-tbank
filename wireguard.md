 Используйте команду wg-quick для активации туннеля:

```sudo wg-quick up wg0```

 Чтобы убедиться, что туннель активен, используйте:

```sudo wg show wg0```

 Чтобы туннель запускался автоматически при загрузке системы:

```sudo systemctl enable wg-quick@wg0```

 Для отключения VPN:

```sudo wg-quick down wg0```


**Для Linux:**

1.  **Переместите файл конфигурации:** Скопируйте ваш файл `wg0.conf` в директорию `/etc/wireguard/`. Возможно, вам понадобятся права суперпользователя:
    ```bash
    sudo mv /path/to/your/wg0.conf /etc/wireguard/
    ```
2.  **Запустите туннель:** Используйте команду `wg-quick` для активации туннеля:
    ```bash
    sudo wg-quick up wg0
    ```
3.  **Проверить статус:** Чтобы убедиться, что туннель активен, используйте:
    ```bash
    sudo wg show wg0
    ```
4.  **Включить автозапуск (необязательно):** Чтобы туннель запускался автоматически при загрузке системы:
    ```bash
    sudo systemctl enable wg-quick@wg0
    ```
5.  **Остановить туннель:** Для отключения VPN:
    ```bash
    sudo wg-quick down wg0
    ```
    


**Для macOS:**

1.  **Откройте WireGuard:** Запустите приложение WireGuard.
2.  **Добавить туннель:** Нажмите "Add Tunnel" (Добавить туннель) в левом нижнем углу окна.
3.  **Импортировать из файла:** Выберите "Import tunnel from file" (Импортировать туннель из файла) и укажите путь к вашему файлу `wg0.conf`.
4.  **Активировать:** Нажмите "Activate" (Активировать) рядом с названием вашего туннеля.
